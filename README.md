# SIMS APP

> School Information Management System

## Build Setup

``` bash
# install dependencies
npm install

# start local project with hot reload at localhost:8080
quasar dev

# serve with hot reload at localhost:8080
quasar play

# build for production with minification
quasar build

# run unit tests
quasar test --unit

# run e2e tests
quasar test --e2e

# run all tests
quasar test
```
